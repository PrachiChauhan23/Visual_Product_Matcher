# Visual Product Matcher

An AI-powered image similarity search system that helps users find visually similar products.

## Features
- Image upload and URL input
- AI-powered similarity matching
- Real-time results
- Category recognition

## Technologies Used
- Python/Flask
- ResNet50
- PyTorch
- SQLite
- HTML/CSS/JavaScript
